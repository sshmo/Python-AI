# Parser

Write an AI to parse sentences and extract noun phrases.

    $ python parser.py
    Sentence: Holmes sat.
            S
       _____|___
      NP        VP
      |         |
      N         V
      |         |
    holmes     sat

    Noun Phrase Chunks
    holmes

## The problem solving approch:

The problem was solved using nltk parser and tokenizer.
